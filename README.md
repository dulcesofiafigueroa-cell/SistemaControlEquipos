# SistemaControlEquipos

## Descripción
Sistema en C para registrar, consultar, modificar y eliminar equipos informáticos usando archivos y control de versiones con Git.

## Funcionalidades
- Registrar equipos
- Mostrar equipos
- Buscar equipos
- Modificar equipos
- Eliminar equipos

## Tecnologías utilizadas
- Lenguaje C
- Git
- GitHub
- VS Code
